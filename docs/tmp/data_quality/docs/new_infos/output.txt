================================================================================
               FORMAT DE SORTIE — RAPPORT EXCEL SIMPLIFIÉ
================================================================================

PRINCIPE
--------

1 ligne = 1 site   (NU_CNT + ID_SITE)
Toutes les anomalies du site sont regroupées dans un seul libellé.


ONGLETS DU RAPPORT
------------------

1. Résumé
   - Statistiques globales (sites actifs, avec anomalies, GPS OK, adresse OK, incomplets)
   - Répartition par code anomalie
   - Répartition par pays

2. Sites avec anomalies
   - 1 ligne par site ayant au moins 1 anomalie
   - Triés du plus grave au moins grave
   - Voir colonnes ci-dessous

3. Données enrichies
   - 1 ligne par site actif (tous les sites)
   - Voir colonnes ci-dessous


COLONNES CLÉS (onglets 2 et 3)
--------------------------------

   NU_CNT              → Identifiant contrat
   ID_SITE             → Identifiant site
   NOM_CLI             → Nom du client
   NOM_SITE            → Nom du site
   PAYS_SITE           → Pays
   CP_SITE             → Code postal
   VILLE_SITE          → Ville
   COORD_X_SITE        → Longitude
   COORD_Y_SITE        → Latitude
   NUM_VOIE_SITE       → Numéro de voie
   RUE_SITE            → Nom de rue
   NUM_RUE_SITE        → Rue complète
   ADRESSE_SITE        → Adresse complète

   _CONTROL_MODE       → GPS_OK / ADRESSE_OK / INCOMPLET
   _CONTROL_STATUS     → OK / A_CORRIGER / A_COMPLETER / A_VERIFIER
   _PRIORITY           → HAUTE / MOYENNE / FAIBLE / OK
   _ISSUE_COUNT        → Nombre d'anomalies sur le site
   _ISSUE_CODES        → Codes anomalies séparés par virgule  (ex: G-01, A-01)
   _ISSUE_SUMMARY      → Libellé consolidé lisible            (ex: GPS manquant | Adresse manquante)


CODES ANOMALIES ET LIBELLÉS
----------------------------

   G-01   GPS manquant
   G-02   GPS incomplet
   G-03   GPS invalide
   G-04   GPS hors pays
   G-05   GPS incohérent avec le code postal

   A-01   Adresse manquante
   A-02   Numéro de voie manquant
   A-03   Rue à vérifier manuellement

   P-01   Pays manquant
   P-02   Code postal manquant
   R-01   Pays non couvert
   D-01   Département / pays à vérifier


EXEMPLES DE LIBELLÉS FINAUX (_ISSUE_SUMMARY)
--------------------------------------------

   "GPS manquant | Adresse manquante"
   "GPS hors pays | Numéro de voie manquant | Rue à vérifier manuellement"
   "GPS incohérent avec le code postal"
   "Adresse manquante"
   "Numéro de voie manquant | Rue à vérifier manuellement"
